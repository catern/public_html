Technical advantages of open source software

  • You can have a single source repository containing both the code providing an API, and the code using that API, allowing easy system-wide refactorings.
  • Relatedly, you don't have to provide a stable plugin API.
  • You can safely ship your software as source code, meaning you don't have to support verification or analysis on compiled binaries. This makes it easy to avoid the "microkernel" antipattern.
  • You and your users can actually run the software.
  • You don't have to ship binaries for many different platforms; package managers will handle that. (Although it's not clear that this one is a purely technical advantage, it might be a social one.)
